Les méthodes sur les tableaux

1. filter

Description : Renvoie un tableau contenant tous les éléments du tableau d'origine qui remplissent une condition donnée (fonction de filtrage).

Exemple :

const numbers = [1, 2, 3, 4, 5, 6];
// Garder seulement les nombres pairs
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // [2, 4, 6]

2. map

Description : Applique une fonction à chaque élément du tableau et retourne un nouveau tableau contenant les résultats.

Exemple :

const numbers = [1, 2, 3, 4, 5];
// Multiplier chaque élément par 2
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

3. find

Description : Renvoie le premier élément du tableau qui satisfait une condition. Si aucun élément ne correspond, retourne undefined.

Exemple :

const numbers = [1, 2, 3, 4, 5];
// Trouver le premier nombre pair
const firstEven = numbers.find(num => num % 2 === 0);
console.log(firstEven); // 2

4. every

Description : Vérifie si tous les éléments du tableau remplissent une condition. Retourne un booléen (true ou false).

Exemple :

const numbers = [2, 4, 6, 8];
// Vérifier si tous les nombres sont pairs
const allEven = numbers.every(num => num % 2 === 0);
console.log(allEven); // true

5. some

Description : Vérifie si au moins un élément du tableau remplit une condition. Retourne un booléen.

Exemple :

const numbers = [1, 3, 5, 6];
// Vérifier s'il existe un nombre pair
const hasEven = numbers.some(num => num % 2 === 0);
console.log(hasEven); // true

6. sort

Description : Trie les éléments du tableau en place. Par défaut, les éléments sont triés comme des chaînes de caractères. Vous pouvez fournir une fonction de comparaison.

Exemple :

const numbers = [10, 5, 20, 15];
// Trier les nombres en ordre croissant
const sorted = numbers.sort((a, b) => a - b);
console.log(sorted); // [5, 10, 15, 20]

7. slice

Description : Retourne une copie superficielle d'une portion du tableau sans modifier l'original. Prend deux arguments : l'indice de début (inclus) et l'indice de fin (exclu).

Exemple :

const numbers = [1, 2, 3, 4, 5];
// Obtenir une sous-section du tableau
const subset = numbers.slice(1, 4);
console.log(subset); // [2, 3, 4]
console.log(numbers); // [1, 2, 3, 4, 5] (inchangé)

8. splice

Description : Modifie un tableau en ajoutant, supprimant ou remplaçant des éléments à un indice donné. Retourne les éléments supprimés.

Exemple :

const numbers = [1, 2, 3, 4, 5];
// Supprimer 2 éléments à partir de l'indice 1
const removed = numbers.splice(1, 2);
console.log(removed); // [2, 3]
console.log(numbers); // [1, 4, 5]

// Ajouter des éléments
numbers.splice(1, 0, 10, 20);
console.log(numbers); // [1, 10, 20, 4, 5]

9. reduce

Description : Applique une fonction à un accumulateur et à chaque élément du tableau (de gauche à droite) pour réduire le tableau à une seule valeur.

Exemple :

const numbers = [1, 2, 3, 4, 5];
// Somme de tous les éléments
const sum = numbers.reduce((acc, num) => acc + num, 0);
console.log(sum); // 15

// Calculer le produit des éléments
const product = numbers.reduce((acc, num) => acc * num, 1);
console.log(product); // 120

Comparaison des méthodes

Méthode Retourne quoi ? Modifie le tableau ?
filter Un nouveau tableau filtré Non
map Un nouveau tableau transformé Non
find Le premier élément trouvé Non
every Un booléen Non
some Un booléen Non
sort Le tableau trié Oui
slice Une copie d'une portion du tableau Non
splice Les éléments supprimés Oui
reduce Une valeur unique Non

Exemple combiné

Pour montrer comment combiner ces méthodes :

const people = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 },
  { name: 'Charlie', age: 35 },
  { name: 'David', age: 40 }
];

// Étape 1 : Filtrer les personnes de plus de 30 ans
const over30 = people.filter(person => person.age > 30);

// Étape 2 : Extraire les noms
const names = over30.map(person => person.name);

// Étape 3 : Trier les noms par ordre alphabétique
const sortedNames = names.sort();

console.log(sortedNames); // ['Charlie', 'David']
